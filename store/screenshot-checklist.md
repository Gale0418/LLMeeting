# Chrome Web Store Screenshot Checklist

Prepare at least these screenshots before submitting LLMeeting to the Chrome Web Store:

1. Side panel idle state with Free badge, provider checkboxes, 基礎辯論, 快速鬥技場 PRO, and 總結辯論 PRO.
2. A running 基礎辯論 with provider status cards showing 回答中 or 已回答.
3. Transcript view with first-round answers and cross-critiques.
4. Free locked message after clicking 快速鬥技場 or 總結辯論.
5. Diagnostics panel expanded to show local provider state.

Recommended capture notes:

- Use a clean Chrome profile or hide personal chat history.
- Use short harmless prompts such as `天為什麼是藍的？`.
- Avoid exposing private account names, emails, or paid-plan information.
