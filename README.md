# LLMeeting 🤖 💬 🤖

想看看不同立場的 AI 怎麼爭論嗎？LLMeeting 讓你在瀏覽網頁時，隨時發起一場 AI 辯論。

🌟 **[👉 前往 Chrome 線上應用程式商店安裝 LLMeeting！](https://chromewebstore.google.com/detail/llmeeting/cjombmcmgifhanenolhaeepbpnlchjdp)**

---

## 🎯 這是什麼？

這是一個好玩的 Chrome 擴充功能！有時候你問 ChatGPT 一個問題，它給了你一個答案，但你可能想知道：「如果是 Gemini 會怎麼說？Grok 會同意嗎？Claude 有沒有不同的見解？」

透過 **LLMeeting** 的側邊欄，你可以一次把問題丟給多個頂尖 AI 模型，然後**讓它們互相看對方的答案、互相吐槽、最後再做總結**！

## ✨ 好玩的功能

- **多方會談**：支援 ChatGPT、Gemini、Grok、Claude 一起加入群聊！
- **交叉詰問**：不只是各自回答，還會讓它們看過對手的答案後，進行 1 到 5 輪的互相評析與辯論！
- **最終裁判**：吵完之後，指定一位 AI 當裁判幫你做最終客觀總結。
- **無縫側邊欄**：在任何網頁隨時呼叫，不用再手動於分頁之間複製貼上到手軟！
